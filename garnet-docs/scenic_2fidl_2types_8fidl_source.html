153 // Rendering target metrics associated with a node.
154 // See also |MetricsEvent|.
155 struct Metrics {
156  // The ratio between the size of one logical pixel within the node's local
157  // coordinate system and the size of one physical pixel of the rendering
158  // target.
159  //
160  // This scale factors change in relation to the resolution of the rendering
161  // target and the scale transformations applied by containing nodes.
162  // They are always strictly positive and non-zero.
163  //
164  // For example, suppose the rendering target is a high resolution display
165  // with a device pixel ratio of 2.0 meaning that each logical pixel
166  // within the model corresponds to two physical pixels of the display.
167  // Assuming no scale transformations affect the node, then its metrics event
168  // will report a scale factor of 2.0.
169  //
170  // Building on this example, if instead the node's parent applies a
171  // scale transformation of 0.25 to the node, then the node's metrics event
172  // will report a scale factor of 0.5 indicating that the node should render
173  // its content at a reduced resolution and level of detail since a smaller
174  // area of physical pixels (half the size in each dimension) will be rendered.
175  float scale_x;
176  float scale_y;
177  float scale_z;
178 };
