20 // Provides rendering target metrics information about the specified node.
21 //
22 // This event is delivered when the following conditions are true:
23 // - The node is a descendant of a |Scene|.
24 // - The node's metrics have changed since they were last delivered, or since
25 // |kMetricsEventMask| transitioned from a disabled state to an enabled state.
26 //
27 // Subscribe to this event to receive information about the scale factors you
28 // should apply when generating textures for your nodes.

31  Metrics metrics;
32 };
