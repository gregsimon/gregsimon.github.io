uint32 MediaPacket::pts_rate_ticks

Indicates the number of PTS ticks that correspond to pts_rate_seconds seconds. In other words, the time represented by a single PTS tick in seconds is pts_rate_ticks / pts_rate_seconds. For example, if PTS is given in nanoseconds, pts_rate_ticks would be 1,000,000,000, and pts_rate_seconds would be 1.
