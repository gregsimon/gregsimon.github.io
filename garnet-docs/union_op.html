scenic ImportResourceOp Op::scenic_import_resource

Interaction with Scenic scene-graph:

Import the specified Scenic resource. Currently only the following resource types are supported:

  • NODE: imported nodes enable the use of scenic.AddChildOp to insert stroke-groups into a Scenic scene-graph.
  • MATERIAL: imported materials can be applied to a stroke-style.
