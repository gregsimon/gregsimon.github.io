Detailed Description

Stores attributes related to a firebase auth token for a given firebase api key. These tokens are obtained by agents using TokenProvider.GetFirebaseAuthToken().

These tokens are minted by the Firebase Auth servers and are meant to be used for authorizing users into Firebase services such as DB and storage.
