Detailed Description

This structure contains additional data that may be present in a Fidl declaration. Some owning contexts may provide some of this data.

string DeclarationData::container_type_key

If this DeclarationData is for an enum or a constant that is contained in a struct or interface, then this field, if populated, is the type_key that refers to the containing struct or interface. This field plays a role inverse to that of |contained_declarations|.

The zero-based ordinal position of this element within its containing scope as it appears in the Fidl declaration. This is not the serialization ordinal.

string DeclarationData::full_identifier

Some implementations may not provide names.

The fully-qualified identifier is the concatenation of the names of the containing scopes, starting from the module name, with a period ('.') between components and with the |short_name| at the end.
