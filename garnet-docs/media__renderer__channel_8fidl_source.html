19  // Establishes the PTS rate used for PTS values in |EngagePacket| and
20  // |RenderRange|. The default PTS rate is |TimelineRate::NsPerSecond|.

23  // Reports packet usage in a manner suitable for video renderers. This method
24  // essentially states that the renderer looked for a packet to render at
25  // presentation time |current_pts| and selected a packet with a PTS of
26  // |packet_pts| and label |packet_label|. If no packet was available for
27  // rendering, |packet_pts| is |MediaPacket::kNoTimestamp| and |packet_label|
28  // is zero. Note that the units reflected by the PTS values is established
29  // using the |PtsRate| method.
30  EngagePacket(int64 current_pts, int64 packet_pts, uint64 packet_label);
31 
32  // Reports packet usage in a manner suitable for audio renderers. This method
33  // essentially states that the renderer needed content for the range starting
34  // at |pts| and ending immediately before |pts + duration|. Note that the
35  // units reflected by the PTS values is established using the |PtsRate|
36  // method.
37  RenderRange(int64 pts, uint32 duration);
38 };
