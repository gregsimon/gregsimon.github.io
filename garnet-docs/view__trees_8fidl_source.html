11 // A view tree is a top-level container for a hierarchy of views.
12 // Each view is intended to operate independently from others and will
13 // generally correspond to discrete interactive spaces such as separate
14 // displays or isolated environments in a multi-user system.
15 //
16 // Within a view tree, certain global invariants may be enforced such as
17 // ensuring that only one view has focus at a time.
18 //
19 // View trees will typically be created by system components responsible
20 // for managing the overall user interface rather than end-user applications.
21 //
22 // LIFECYCLE
23 //
24 // Use |ViewManager.CreateViewTree()| to create a view tree. The client
25 // uses the |ViewTree| interface to manage the view tree's content
26 // and implements the |ViewTreeListener| interface to handle events.
27 //
28 // To destroy a view tree, simply close the |ViewTree| channel.
29 //
30 // SETTING A ROOT VIEW
31 //
32 // Use |GetContainer()| to obtain an interface for manipulating the root view.
33 //
34 // See |ViewContainer| for more information.
35 //
36 // GETTING SERVICES
37 //
38 // The view tree's |app::ServiceProvider| offers access to many services
39 // which are not directly expressed by the |ViewTree| interface itself, such
40 // as input, accessiblity, and editing capabilities.
41 //
42 // For example, perform the following actions to dispatch input events:
43 //
44 // 1. Call |GetServiceProvider()| to obtain the view's service provider.
45 //
46 // 2. Ask the service provider for its |InputDispatcher|.
47 //
48 // 3. Send input events to the dispatcher for delivery to views.
49 interface ViewTree {
50  // Gets the view tree's token.
51  GetToken() => (ViewTreeToken token);
52 
53  // Gets a service provider to access services which are associated with
54  // the view tree such as input, accessibility and editing capabilities.
55  // The view tree service provider is private to the view tree and should
56  // not be shared with anyone else.
57  //
58  // See |InputDispatcher|.
59  GetServiceProvider(app.ServiceProvider& service_provider);
60 
61  // Gets an interface for managing the view tree's root.
62  GetContainer(ViewContainer& container);
63 };
